# C++ Sorting Algorithms

This repository contains **implementations of fundamental sorting algorithms in C++**.  
The code is written with a focus on **clear logic, simplicity, and understanding core algorithmic concepts**.

These implementations are suitable for **students learning Data Structures and Algorithms** and for practicing basic C++ programming.

---

## Sorting Algorithms Included

- Bubble Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort

Each algorithm is implemented using **simple array manipulation** and does not rely on advanced C++ features.

---

## Key Features

- Pure C++ implementations
- Beginner-friendly code structure
- Focus on algorithm logic
- Easy to read and reuse
- Suitable for academic and learning purposes
